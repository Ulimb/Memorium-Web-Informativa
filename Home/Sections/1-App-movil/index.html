Memory game

Memory Game es una herramienta de evaluación y entrenamiento cognitivo diseñada para mejorar la memoria visual a corto plazo y la atención. A través de la presentación de imágenes secuenciales, el usuario pone a prueba su capacidad para recordar y comparar visualmente, estimulando áreas del cerebro relacionadas con la retención de detalles y la concentración.

¿Cómo funciona?

Menú principal: Al iniciar el juego, se accede a un menú con diversas opciones:

  • Instrucciones: Explica el objetivo del juego, que consiste en recordar y comparar imágenes.
  • Botón de audio: Opción para escuchar las instrucciones.
  • Botón de información: Permite conocer los beneficios de practicar con Memory Game.
  • Dificultad: Selección entre niveles fácil, intermedio y difícil.
  • Opciones avanzadas: Ajuste de tiempos de visualización de imágenes y distractores para añadir dificultad.
  • Comenzar: Al presionarlo, se muestra una lista de categorías para que el usuario elija y el juego comience.

Dinámica del Juego

  1. Primera imagen: Se muestra una imagen con el mensaje "¡Recuerda esta imagen!" durante el tiempo establecido en las opciones avanzadas.
  2. Distractor: Luego de visualizar la imagen, aparece un distractor durante el tiempo previamente seleccionado, incrementando la dificultad del juego.
  3. Segunda imagen: Aparece una nueva imagen acompañada de la pregunta "¿Es la misma imagen?". El usuario debe decidir si la imagen es la misma que la primera y responder "es la misma" o "es diferente".
  4. Retroalimentación: Después de la respuesta, se muestra al jugador si acertó o no, junto con ambas imágenes para referencia. Finalmente, se ofrece la opción de jugar de nuevo o finalizar.

Numerium

Numerium es un juego de secuencias numéricas diseñado para mejorar la memoria de trabajo numérica y verbal, así como la atención sostenida. A través de la memorización y reproducción de secuencias de números, el usuario fortalece su capacidad para recordar y procesar información de manera efectiva en el corto plazo.

¿Cómo funciona?

Menú principal: Al iniciar el juego, el usuario tiene acceso a varias opciones:

  • Instrucciones: Explicación del objetivo del juego: recordar y repetir secuencias de números en el orden correcto.
  • Botón de audio: Opción para escuchar las instrucciones.
  • Botón de información: Explica los beneficios específicos de jugar a Numerium.
  • Ver tutorial: Permite ver un tutorial sobre el funcionamiento del juego.
  • Opciones avanzadas: Permite ajustar el tiempo de visualización de los dígitos, el intervalo entre ellos y la cantidad total de dígitos a memorizar.
  • Comenzar: Inicia el juego con la configuración seleccionada.

Dinámica del Juego

  1. Visualización de dígitos: Se muestra un número (entre 0 y 9) junto al mensaje "Recuerda este número!".
  2. Distractor: Tras cada número, se muestra un distractor según el tiempo configurado en opciones avanzadas.
  3. Ingreso de respuesta: Al finalizar la secuencia, el usuario debe ingresar los números en el orden correcto.
  4. Retroalimentación: El usuario dispone de hasta cinco intentos. Si la respuesta es correcta, recibe una confirmación y opción para jugar de nuevo o salir. Si falla en los cinco intentos, se muestra la secuencia correcta.

Contrarium

El juego Contrarium está diseñado para desafiar y mejorar su memoria semántica y control inhibitorio. A través de una dinámica de selección de antónimos, se invita al usuario a identificar la palabra con significado contrario entre cuatro opciones presentadas.

¿Cómo funciona?

En el menú principal, se encuentran las siguientes opciones:

  • Instrucciones: Explicación del objetivo del juego, que consiste en seleccionar el antónimo de una palabra dada entre varias opciones.
  • Botón de audio: Opción para escuchar las instrucciones en formato de audio.
  • Información: Detalle sobre los beneficios de entrenar con Contrarium.
  • Comenzar: Al presionar este botón, el juego inicia automáticamente.

Dinámica del Juego

Al iniciar el juego, aparece una palabra en la esquina superior izquierda, y cuatro opciones de respuesta en el centro de la pantalla. Una de estas opciones es el antónimo de la palabra principal, mientras que las otras tres son sinónimos.

El usuario debe seleccionar el antónimo de la palabra indicada. En caso de duda, puede usar la bomba de ayuda, que eliminará una o más opciones incorrectas hasta que queden dos alternativas.

La ayuda con la bomba solo se puede usar mientras haya más de dos opciones en pantalla. Si el usuario selecciona la respuesta incorrecta, puede intentarlo nuevamente hasta dar con el antónimo correcto. Al acertar, aparecerá una pantalla de felicitación y las opciones de volver a jugar o finalizar.
